# Toggle

A two-state button that can be either on or off.

This higher order component can transform any `Button` or `IconButton` into a toggle button.

It manages `aria-pressed` attribute for a11y, but it's up to you to adapt the styles to reflect the state.

## Install

```sh
npm install @spark-ui/toggle
```

## Import

```tsx
import { Toggle } from '@spark-ui/toggle'
```
