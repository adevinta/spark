# FloatingLabel

Renders an accessible and animated label associated with controls. **It doesn't work on its own, it must be wrapped by a form element controlling when it has to be expanded.**

## Install

```sh
npm install @spark-ui/label
```

## Import

```tsx
import { FloatingLabel } from '@spark-ui/label'
```
